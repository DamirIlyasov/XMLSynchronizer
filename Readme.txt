Здравствуйте.


1)Заполните все файлы .properties
Для логов создан файл в директории resources/files под названием logs.xml
В log4j.properties советую указать путь к нему.


2)Для корректной работы приложения перед запуском либо в командной строке пропишите
mvn clean install -Pdb-changelog -Pseeds
Либо соберите проект через Maven, поставив галочку напротив Profiles с именами
db-changelog и seeds.
(Это загрузит базу данных и информацию в неё)



3) Далее запускать метод main класса Main.
Для синхронизации в директории resources/files есть файл data.xml, советую указывать
путь к нему при работе.
